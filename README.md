# The Wilson Partners Website

Professional marketing site for **The Wilson Partners**, a leadership advisory firm focused on responsible, human-centered AI.

Built with Next.js 16, TypeScript, and Tailwind CSS.

## Pages

- **Home**: Hero, value proposition, service overview
- **Services**: BTG Works-inspired detailed service sections with anchor navigation
- **About**: Firm story, values, team, and client work
- **Contact**: Direct contact info and inquiry form

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customizing Content

All copy lives in `src/lib/content.ts`. Update contact details, services, and bio there.

## Deploy

Deploy to [Vercel](https://vercel.com) or any Node.js host:

```bash
npm run build
npm start
```

## Note on Contact Info

Email and phone in `src/lib/content.ts` are placeholders. Replace with real contact details before going live.
